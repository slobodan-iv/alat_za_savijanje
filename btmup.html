Моделирање склопа "Bottom-Up" поступком

Код "Bottom-Up" поступка моделирања склопа, моделирамо посебно компоненте, које после убацимо у склоп.

Ознака за месинг је застарела ознака. Новије ознаке су: , тј.

Материјал рукавца је Č.0545 по старој ознаци (стандарду), а по новом стандарду ознака је: 1.0050 (Е295)

Материјал тискача је Č.4840 по старој ознаци (стандарду), а по новом стандарду ознака је: 1.2063 (145Cr6)

Материјал стубних вођица је Č.1220 по старој ознаци (стандарду), а по новом стандарду ознака је: 1.0401 (C15)

Материјал чаура је P.CuSn14.01

Материјал опруге је Č.0545 по старој ознаци (стандарду), а по новом стандарду ознака је: 1.0050 (Е295)

Каталог опруга - download

Материјал држача са водиљом опруге је Č.1120 по старој ознаци (стандарду), а по новом стандарду ознака је: 1.0301 (C10)

Материјал сегмента калупа је Č.4321 по старој ознаци (стандарду), а по новом стандарду ознака је: 1.7147 (20MnCr5)
